# POML Lab Website (GitHub Pages)

This repository is configured for a **database-free** GitHub Pages website using Markdown/YAML content files.

## Quick start
1. Push this repo to GitHub.
2. In GitHub repository settings, enable **Pages** from the default branch root.
3. Add your real logo at `assets/images/logo.png` (top-left header logo).

## Content management (manual admin)
Update these folders and push commits:
- `_people/`
- `_publications/`
- `_news/`
- `_gallery/`

The `admin/` page inside the website documents the same workflow.
